## **Getting Started**

## Installation

> kyunghun branch를 Clone 합니다.

```sh
git clone -b kyunghun --single-branch https://github.com/Wanted-07-team-9/pre-onboarding-7th-2-1-9.git

# npm으로 설치 진행
pre-onboarding-7th-2-1-9 %  % npm install
```

- 프로젝트 실행 및 빌드 방법

```sh
# Local Dev Server 실행
```
